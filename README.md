## Getting Started

## Prerequisites

You'll need to have a basic environment with React Native installed and back-end running. 

**Installing dependencies**
```
$ npm install
```

## Routes

The base URL is: http://localhost:3000/home-decor/api/
The endpoints: 

- /user
- /post
- /categories
